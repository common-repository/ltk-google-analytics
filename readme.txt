Integrate Google Analytics with your WordPress and enable traffic tracking.

== Description ==

With this plugin you will be able to integrate the Google Analytics code in your website. It also enables you to track additional data such as authors, categories and tags with Google Analytics dimensions.

For controlling its settings, the plugin creates a submenu in the settings entry instead of a completely new menu entry.

List of features:

* Adds the Google Analytics tracking code to your website.
* Use of custom dimensions to track author, category and tag statistics.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ltk-google-analytics` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress and you're done.
